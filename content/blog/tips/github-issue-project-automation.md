# In commit message just add following text as format #<issue number> followed by fix, close

- `fix #xxx`
- `fixes #xxx`
- `fixed #xxx`
- `close #xxx`
- `closes #xxx`
- `closed #xxx`
- `resolve #xxx`
- `resolves #xxx`
- `resolved #xxx`
